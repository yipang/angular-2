Angular Data Request Practice

Instructions
  1. Create an angular module called myApp in your JavaScript code (documentation)
  2. Add a contrller to your module called customersCtrl, whose execution function takes in two parameters: $scope and $http
  3. Make an $http.get request to the following URL (documentation): 
    http://www.w3schools.com/angular/customers.php
  4. When your $http.get request is successful, set the variable $scope.names equal to the records in the response's data.
  5. In your HTML, bind your application to the <div> below using the ng-app and ng-controller directives.
  6. Inside the <ul> below, use the ng-repeat directive to create an <li> for each element in your $scope.names variable
  7. In each <li> that you create, use an angular expression to show some information about each record
  • {{ a.name }}
