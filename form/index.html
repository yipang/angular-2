Instructions
  1. Initiate your angular application myApp on the <div> element that encapsulates your form. Make sure to specify your controller as myCtrl
  2. Give the <form> element a name property and set it to "myForm"
  3. Add a property novalidate to the <form> element (we'll do our own validation)
  4. Give your Name input a name property and set it equal to "username" so it can be referenced.
  5. Let's add some form requirements: use ng-minlength="SOME-NUMBER" to set a minimum length of 6 for the username input.
  6. Add this <p> element to serve as a warning if the username input's length is too short 
    <p ng-show="myForm.username.$error.minlength" class="help-block">Username is too short.</p>
  7. Using a similar approach, include an error message for your email input if it does not have a valid email (hint: there is an email property to the $error variable.
  8. Using the ng-disabled directive, disable the submit button if the form is invalid (hint: FORMNAME.$invalid is a boolean value of the form's validity)
  9. Add a conditional class "has-error" to each div with class form-group field if your field is has been touched and is invalid. Hint: 
    ng-class="{'DESIRED_CLASS' : EXPRESSION TO BE EVALUATED}"
  10. Add an ng-click directive to the submit button that executes the submit function from your JS code.
